Pure CSS Curves

These curves are created with pure CSS, and the section are containing an empty DIV where the magic happens.

The curves are made in the sudo selectors ::before and ::after.

It is not a perfect solution, as you can see, the curves are not lining up perfect, and it is quite a fiddly job to make something that looks ok.

Curves Made With SVG

Here we are using a SVG as inline code.

I have made a DIV with the SVG code, and then use a position absolute to place the divider.

The SVG are made at Shape Divider, where you can make your own custom dividers, and download them as inline code or as an image file.

SVG Images Used as Spacer
